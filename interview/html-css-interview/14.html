响应式布局常见方案

响应式布局常见方案

通过媒体查询为不同尺寸屏幕设置不同的根节点font-size，使用用rem即可

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    @media only screen and (max-width: 374px) {
      html{
        font-size: 88px;
      }
    }
    @media only screen and (min-width: 375px) and (max-width: 413px) {
      html{
        font-size: 98px;
      }
    }
    @media only screen and (min-width: 414px) {
      html{
        font-size: 108px;
      }
    }
    .odiv{
      font-size: 0.16rem;
    }
  </style>
</head>
<body>
  <div class="odiv">
    21日下午，县政协党组书记、主席肖吉武到六合垸管理区曹夹湖社区宣讲党的十九届六中全会精神。肖吉武从党的十九届六中全会基本概况、主要内容、如何贯彻落实党的十九届六中全会精神三个方面，对全会精神进行了系统的解读和深入阐述。他强调，全体党员干部要继续深入学习党的十九届六中全会精神，加大宣传力度，把全会精神传达下去。要结合管理区实际情况，不断补齐短板，保质保量完成年度目标任务。要严格落实疫情防控各项部署，筑牢疫情防控安全防线，为群众生命健康提供有力保障。要扎实推进平安稳定工作，着力解决影响农村安全稳定的突出问题，提升人民群众的获得感、幸福感、安全感。
  </div>
</body>
</html>
