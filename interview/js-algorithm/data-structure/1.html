栈

栈

后进先出的数据结构；像一个桶，先放进去的物品最后最后才能拿出来，最后放入的物品可以首先拿出来。下面用javascript的数组方法模拟栈

let arr = []
arr.push(0)
arr.push(1)
console.log(arr) // [0, 1]
arr.pop()
console.log(arr) // [0]
arr.pop()
console.log(arr) // []

class Stack {
  constructor () {
    this.arr = []
  }
  push (item) {
    this.arr.push(item)
  }
  pop () {
    this.arr.pop()
  }
  peek () {
    return this.arr[this.arr.length - 1]
  }
}
